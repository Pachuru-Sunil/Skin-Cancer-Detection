# Skin Cancer Detection using Deep Learning

## Overview
A deep learning model for classifying seven types of skin lesions using the HAM10000 dataset and EfficientNetV2B0.

## Technologies Used
- Python
- TensorFlow
- Keras
- Google Colab
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Dataset
HAM10000 Dataset

## Model
EfficientNetV2B0 (Transfer Learning)

## Results
- Training Accuracy: 91%
- Validation Accuracy: 90%
